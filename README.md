# New JavaScript Project

Coding exercise Game Of Life for the Test Driven Development Mooc course in Helsinki/Open University Spring 2023.

## Prerequisites

You'll need a recent [Node.js](https://nodejs.org/) version. Then download this project's dependencies with:

    npm install

## Running 

Run script with RLE file

    npm start [file path] [step count]

## Developing

Run tests once

    npm run test

Run tests continuously

    npm run autotest

Code reformat

    npm run format
